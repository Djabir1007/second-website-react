react query - это библиотеки для управления состоянием сервера
суть этой библиотеки в том,чтобы упросить делание запроса на сервер

npm i @tanstack/react-query - с помощью этой команды устанавливаем react-query
хук useQuery() - позволяет легко получать данные с сервера и управлять их состоянием
все наше приложение оборачиваем в QueryClientProvider,у которого есть обязательный атрибут client,тем самым мы связываем наше приложение с экземпляром QueryClient
Теперь нам можно в любых компонентах использовать useQeuery()

queryKey -это уникальный ключ запроса, по которому react query кэширует и находит данные
queryFn - функция,которая должна возвращать промис с данными

также изучил axios - он нужен для выполнения асинхронных запросов
